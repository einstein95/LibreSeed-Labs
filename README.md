# LibreSeed Labs

![LibreSeed Labs Banner](https://raw.githubusercontent.com/Official-Husko/LibreSeed-Labs/refs/heads/main/textures/app_icons/app_icon_432.png)

---

LibreSeed Labs is an open-source Godot game project, rebranded and rebuilt from the original Steam game "Upload Labs." This project provides a privacy-respecting, mod-friendly alternative to the original, free from ads, telemetry, and premium paywalls.

- **No Ads or Telemetry**
  - All advertising and tracking code has been removed.
- **No Premium Paywalls**
  - All premium features are unlocked; no microtransactions or paid upgrades.
- **Modding Support**
  - The full source code is available, allowing anyone to create modded builds or contribute improvements.
- **No Steam Dependency**
  - The game runs independently of Steam and its DRM.
